苹果电脑制作OS X 10.10 USB启动安装盘

2015年4月29日
| it-maintenance

1. 准备工作

把OS X Yosemite 正式版的安装程序（“Install OS X Yosemite.app”，显示为 “安装 OS X Yosemite.app”） 移动到“应用程序”文件夹。

2.  格式化U盘

插入U盘（至少8GB） → Finder → 应用程序 → 实用工具 → 磁盘工具 → 左侧选中该U盘 → 右侧点击“分区” → “分区布局”选1个分区 → “分区名称”输入OSXINST →"格式”选择Mac OS 扩展（日志式） → “选项"选择GUID 分区表 → 点击"应用”，等待格式化完成。

3. 制作启动安装盘

Finder → 应用程序 → 实用工具 → 终端 → 输入以下命令

sudo /Applications/Install\\ OS\\ X\\ Yosemite.app/Contents/Resources/createinstallmedia --volume /Volumes/OSXINST --applicationpath /Applications/Install\\ OS\\ X\\ Yosemite.app --nointeraction

回车后输入管理员密码，等待制作完成（会显示Copy complete，Done）。

补充：开机按住option按键不放，会出现启动磁盘选择界面，此时就可选U盘安装系统。
