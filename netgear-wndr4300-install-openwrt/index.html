NETGEAR WNDR4300安装OpenWRT

2015年4月10日
| it-maintenance

1. 下载对应的OpenWRT固件

WNDR4300的CPU是Atheros AR9344，目前最新固件为barrier_breaker-14.07-ar71xx。

OpenWRT固件有两种：一种是factory固件（用于从原厂系统升级），一种是sysupgrade固件（用于从OpenWRT系统升级）。

2. 在WNDR4300原厂系统安装OpenWRT

用网线连接电脑和路由器的LAN接口，打开浏览器进入路由器原厂web管理界面（默认http://192.168.1.1账号admin密码password），找到“固件升级”，上传之前下载的openwrt-ar71xx-nand-wndr4300-ubi-factory.img并安装。路由器在安装完成后会自动重启。

3. 设置OpenWRT

进入OpenWRT的web管理界面（默认http://192.168.1.1账号root密码为空），立刻修改root的密码，然后设置只允许从br-lan端口进行SSH登录。

用root账号通过SSH登录路由器，执行以下命令：

opkg update
opkg install luci-i18n-chinese
opkg install openssh-sftp-server

说明：系统升级（需要较长时间，耐心等待）；安装中文管理界面；安装sftp服务。

系统刷成OpenWRT后可以安装其他服务，比如OpenWRT安装OpenVPN客户端和OpenWRT安装Shadowsocks和ChinaDNS。
