WIN7刷新图标缓存的批处理

2015年3月18日
| it-maintenance

WIN7下面，把以下命令保存为批处理文件，用管理员权限执行即可立刻刷新图标缓存。

rem 关闭 explorer.exe
taskkill /f /im explorer.exe
attrib -h -i %userprofile%\\AppData\\Local\\IconCache.db
del %userprofile%\\AppData\\Local\\IconCache.db /a
rem 打开 explorer.exe
start explorer
