暗黑破坏神4 DLC2 心得

2026年5月1日
| game

暗黑4 DLC2对玩法有较大调整，但游戏内部缺乏相关的说明指引，全靠玩家自己摸索，所以有必要整理总结一份心得。

目前针对S14赛季，已根据3.1.1a补丁调整更新内容。

一、通用心得

1、过剧情难度随意，可以选普通看剧情。 有一个号完整过了剧情（中途换号得重来），新建其他号就可以跳过剧情。

2、由于DLC2包含了技能分支、做装系统等一堆东西，要完整体验赛季必须买DLC1+2合集（只是看看剧情可以不买DLC）。DLC2新主城也适合设置为收藏，功能齐全。

3、练级过程中紫色和橙色军械贡品 看到不要打都留着，紫色军械贡品折磨X（T10）以上再打高概率出神话暗金，5个橙色军械贡品合成1个紫色军械贡品。

4、洗套装神符需要大量的神符和封印分解材料（遇到就捡），橙色封印无用的直接卖钱（很值钱），暗金神符留着（暗金做成暗金神符需要任意3个暗金神符）。

5、折磨难度改为T1~T12后，T8及以上才有较好掉落，开荒期优先提升折磨难度（选对开荒职业很重要），战争计划也要优先升级。T10开始所有物品都会掉落（全解锁），更高难度只是提高掉率。

6、赛季开荒围绕战争计划和赛季任务进行，赛季玩法如果奖励够多再去刷。战争计划给予大量的各种奖励，赛季任务奖励还包括关键的技能点和巅峰点（必须拿全）。前期提升人物等级最重要，装备也只看装备等级有啥用啥（同时记得回火和附魔提高关键伤害）；神符套装和威能有啥用啥（T8之后专门搞），开荒BD前期不用管暗金（T8之后专门搞）；雕文等级不用刷（T10之后考虑），巅峰不开雕文插槽直奔传奇节点（后期再调整巅峰盘插雕文）。

7、缺乏资源解决办法汇总：

  • 缺威能：地下城没有保底威能了，传奇威能缺乏的可以用魔盒定向类别做橙装分解。
  • 缺宝石：刷带宝石词缀的梦魇符印。巨型宝石和更高级的赫拉迪姆宝石不要分解，系统都只会给10万碎片（等于皇家宝石的分解）非常亏，留着和别人交换。
  • 缺金币：刷带金币词缀的梦魇符印。
  • 缺遗忘之魂：可以去刷材料词缀的梦魇符印，或者去刷军团。
  • 缺做装底材：古币拿去赌所需部位的做装底材（DLC2主城不会出暗金，其他主城会）。
  • 缺巢穴钥匙：可以多刷下地狱狂潮（开箱和刷蠕虫）。
  • 缺强力巢穴钥匙：刷初级巢穴首领。
  • 缺精良巢穴钥匙：刷高级巢穴首领。
  • 缺同调粉尘（用于重置暗金威能）：战争计划-梦魇地下城点出信念箱子，刷梦魇地下城开信念箱子（中途不能回城，会影响结算）。
  • 缺普通嬗变材料：刷材料词缀的地下城、材料贡品的幽暗之城。
  • 综合收益：连打三关的升级梦魇速刷。

二、赛季特色

S14赛季

赛季主题【死亡苏醒】，两个新玩法：一个是混沌裂隙，一个是神话暗金3.0。

没有赛季能力，赛季BOSS夺魂者（掉落各种神话暗金和升级材料）。

赛季期限：2026年7月-9月。

混沌裂隙

  • 世界各地寻找裂隙：普通裂隙不会出现异界行者，地狱狂潮裂隙概率出现异界行者，巨型裂隙必定出现异界行者。
  • 打败异界行者会出现蓝门，进入蓝门密室完成任务获取精良巢穴钥匙。
  • 带有裂隙词缀的梦魇地下城内关闭一定数量裂隙也会出现蓝门。
  • 赛季BOSS有最高的概率掉落神话暗金及神话暗金升级材料（混沌界碎片）。

暗金相关

  • 普通暗金有对应的神话暗金，可以通过魔盒制作。
  • 自制的神话暗金只能装备一件（后续补丁取消了限制），掉落的不限制。
  • 标志性神话暗金（老的神话暗金）特效大幅削弱，和暗金神话相比没有特别优势。
  • 附魔师可以给普通暗金/神话暗金/标志性神话暗金附魔修改一条词缀。

其他调整

  • 神话封印只有2种6插槽。
  • 宝石嬗变改为提升20~30%，基本废掉了。
  • 战争计划组队发起投票，通过后生成新的战争计划全队一致。战争计划不同难度节点给的经验有差别。
  • 新增独狼模式（SSF）完全单人游玩，没有组队没有交易。
  • 天塔通关有奖励了（100层以上+1000名以内）：外观、头衔、装备，赛季结束后获得徽章。
  • 各职业平衡：S13赛季主流玩法大幅削弱，加强非主流玩法，比如电球巫师GG。
  • 墨菲斯托回响转阶段过场动画现可跳过。
  • 古币上限2500改为25000，金币上限提高至999999999999。

S14赛季整体评价

先说优点：

  • 新增SSF独狼模式，利好纯单机玩家，不用和组队BUG玩家呆在一起。
  • 暗金可以附魔洗一个词条。
  • 宝石可以刷宝石词缀地下城获取，没那么脑残了，当然后续要砍了掉落另说。

再说缺点：

  • 全职业被削，很难站得住，回到魂斗罗玩法。
  • 没有赛季能力，赛季玩法非常空洞，傻乎乎原地罚站面对各种地板技（联动防御端被大削负面体验拉满）。
  • 赛季BOSS掉落很差，没几个人去刷，神话暗金刷别的就行。
  • 附魔洗词缀消耗金币大幅增加，然后900%精进手段刷金币被砍没了，洗神话暗金的附魔词缀洗到破产。
  • 玩永恒服资源充足，比赛季服体验更好（赛季玩法和赛季BOSS没刷的必要+资源获取大幅削弱+没有赛季能力），突出一个本赛季可有可无。
  • 神话暗金掉率大幅被砍+没有刷指定掉落的地方+词条随机。很难掉就算了，巨大的神话暗金池完全随机出，就算出了大部分还都是垃圾（天坑）。
  • 自己做神话暗金同部位随机（天坑），自己做的只能装备一个（不限制还能挽回点）。 搞笑的是自制普通暗金（白装升级/三合一）、先祖暗金做成暗金神符也是只能装备一个，和自制神话暗金不能并存，做装系统基本废了。
  • 这几条联动效果就是本赛季刷神话暗金非常劝退，没有爽感。

整体评价：拉跨，属于凑数赛季。

进入瓶颈期，就可以去玩别的游戏，有空回来看看：

  • 如果后续改了脑残设定，可以接着刷。
  • 如果一直没改，下个赛季再看。
  • 3.1.1补丁提高了神话装备和钥匙掉率，而且各BOSS的暗金掉落池也可以出对应的神话暗金，缓解了一部分问题。
  • **3.1.1a补丁移除了自制装备只能装备一件的限制，这个赛季算是可以接着玩了。

S15赛季前瞻

在S15赛季的魔盒中，会移除神话暗金的重铸机制，并新增普通暗金直接升级神话暗金的机制，该机制会保留原有的普通暗金。

三、BOSS和暗金掉落

1、BOSS分类

  • 初级巢穴首领（巢穴钥匙 开箱）：瓦尔申、格里戈利、冰中野兽、齐尔领主、乌里瓦尔。
  • 中级巢穴首领（强力巢穴钥匙 开箱）：督瑞尔、安达莉尔、憎恨先驱、屠夫。
  • 上级巢穴首领（精良巢穴钥匙 开箱）：彼列、夺魂者。
  • 初级和中级BOSS打完，随机出现彼列伏击，击败概率掉落精良巢穴钥匙。
  • 阿斯塔洛斯：升级梦魇地下城（升级符印 开启）打完三关后的最终BOSS。
  • 巴图克：炼狱魔潮（炼狱魔潮罗盘 开启） 完成所有轮次花费666点数召唤。
  • 墨菲斯托（伪先知的奥义 开箱）：紫色封印和神符掉率高一点，开箱的钥匙去刷高级巢穴首领和地狱狂潮开箱。

2、神话暗金

  • 所有BOSS都有概率掉落神话暗金，彼列和夺魂者掉落神话暗金的概率高于其他BOSS。
  • 铁匠可以用2个精美电花合成1个随机标志性神话暗金（老的神话暗金）。
  • 首饰匠可以制作指定部位的随机神话暗金（普通暗金的神话版本）。
  • 魔盒可以把普通暗金升级为同部位随机神话暗金（说是升级，实际就是同部位随机变个出来，不会继承底材暗金的任何属性），单手武器的主副手都算一个部位。
  • 用魔盒可以把白装升级为普通暗金，也可以同个三合一生成同个普通暗金/神话暗金/标志性神话暗金，但是都算自制。
  • 自制的装备（普通暗金/神话暗金/标志性神话暗金/暗金神符）只能装备一件，掉落的不限制。

  • 谐角之冠（全职业 头盔）
  • 堕狱传承（全职业 头盔）
  • 安达莉尔的仪容（全职业 头盔）
  • 塞利格的溶解之心（全职业 项链）
  • 无星夜空之戒（全职业 戒指）
  • 虚假死亡之衣（全职业 胸甲）
  • 泰瑞尔之力（全职业 胸甲）
  • 末日使者（除灵巫外 单手剑）
  • 艾尔度因，正义之剑（除灵巫外 单手剑）
  • 阿瓦里昂，莱姗德之矛（德鲁伊、巫师 双手杖）

2）神话暗金（普通暗金神话版）

  • 神话暗金必定为先祖品质，暗金威能的效果提升30%，所有词缀必定取最大值。
  • 神话暗金从附魔、嬗变和回火中添加词缀时，始终为满值。
  • 可以通过魔盒把普通暗金制作成同部位随机神话暗金（单手武器、副手和盾牌共享同一个部位类别），刷赛季BOSS获取升级材料。

3、BOSS掉落的暗金

BOSS的暗金掉落池由【通用暗金掉落池】+【专属暗金掉落池】组成。

除了彼列，每个BOSS有自己的暗金掉落池。

各BOSS的首领战利品可以直接用魔盒融合，5个相同首领战利品 合成 1个该首领池子里的随机暗金。

四、战争计划

战争计划优先选择橙色、黄色节点，都没有再选打BOSS和其他的。橙色节点奖励是神符和封印的时候有可能出现紫色神符和封印。

没有好的节点可以重置战争计划（上限3次），每完成一轮战争计划恢复到可重置3次。

战争计划极小概率出现紫色节点，紫色节点奖励是神符和封印的时候更高可能出现紫色神符和封印，紫色节点奖励是宝石的时候掉5个顶级宝石。

1、密语之树

  • 中路【朴素赠礼】-【材料财富】-【混沌垂青】。
  • 右路【残酷报偿】。
  • 左路点到底，种树分支。

2、梦魇地下城

  • 左路点到【无畏信念】，开信念箱子。
  • 中路点到底，刷三连地下城（升级梦魇符印）每层打完可以三选一，容易选出宝石金币地精这些。
  • 最后点出右路【地精掉宝】。
  • 升级梦魇符印用5个普通梦魇符印可以合出来。

3、地狱狂潮

  • 中路点到【地狱之口】，极大提高刷狂潮效率。
  • 右路点到【不灭余烬】，死了不会损失余烬。
  • 右路点出【地狱奖赏】，一次性消耗666，不用到处找箱子开了。
  • 中路点出【爆裂幼虫】，额外获取首领战利品。

4、幽暗之城

  • 中路点到底，任意地方都会掉落先祖军械贡品，5个先祖军械贡品可以合成1个神话军械贡品。
  • 左路只点到【沟渠污物】，最后一级的宝箱开启会消耗神话军械贡品，非常亏。
  • 最后一点右路。

5、巢穴首领

  • 中路点到底，身上随时留3个精良巢穴钥匙才能开最终红门。
  • 左路点到【丰饶巢穴】，击败BOSS后的奖励箱子+1。
  • 剩下点数看需要，点要经常刷的BOSS。

6、炼狱魔潮

  • 中路点到【喧扰】，混沌浪潮分支出现以太领主和地精很容易刷高分。
  • 左路点到【炼狱秘宝】，1000点开超级箱子（刷的高分有地方消耗）。
  • 最后一点中路【厄运将至】。

7、魔渊

  • 中路点到【克劳恩的恩赐】，加一次雕文升级机会。
  • 左路点出【岩石之心】，带着岩石打怪概率掉进度宝珠。
  • 左路点出【该死的窃贼】，随机刷掉进度宝珠的窃贼。
  • 最后看需要点中路或者右路，升级雕文就点出【克罗恩的灵魂】。

五、赫拉迪姆魔盒

1、6种棱柱对应的词缀：

不是所有词缀都能通过魔盒调整，魔盒无法调整的词缀需要底材自带或者避开。

附魔改过的词缀无法通过魔盒调整，可以利用这点锁住要保留的词缀（附魔选择不改动词缀内容）。

1）攻击类（侵略棱柱）

  • 武器伤害
  • 核心属性（比如巫师是智力）
  • 荆棘
  • 各类伤害倍增（含全伤害倍增，都是独立乘区）
  • 攻击速度
  • 暴击几率

2）防御类（保护棱柱）

  • 生命上限
  • 生命回复
  • 护甲值
  • 全元素抗性
  • 各种元素抗性
  • 伤害减免
  • 闪躲几率
  • 击杀回复生命
  • 击中回复生命
  • 幸运一击回复生命

3）资源类（资源棱柱）

  • 资源上限
  • 资源回复
  • 资源消耗减免
  • 资源生成
  • 击杀回复资源
  • 击中回复资源
  • 幸运一击回复资源

4）机动和功能类（实用棱柱）

  • 移动速度
  • 最大闪避次数
  • 攻击缩短闪避冷却时间
  • 闪避后提升移动速度
  • 冷却时间缩减

5）技能和核心属性类（娴熟棱柱）

  • 特定技能或技能类别等级（不含全技能）
  • 核心属性（比如巫师是智力）

6）抗性类（多彩棱柱）

  • 物理抗性
  • 全元素抗性
  • 各种元素抗性
  • 洗全抗可以利用全抗和单抗不能并存来操作：先加单抗/装备自带单抗，再加抗性词缀就是全抗，然后移除一条抗性词缀50%概率全抗留下，没留下全抗再来一遍。

2、装备改造

1）添加词缀

  • 底材有最想要的太古词缀再来补其他词缀。
  • 把黄蓝白装词缀添加到4个，达到最大词缀数量。
  • 添加定向棱柱调整想添加的词缀类别。
  • 配合删除词缀调整，只剩下一个词缀需要调整也可以用附魔来改。
  • 不会出星号词缀。

2）随机重洗

  • 一条随机词缀换成其他类别的词缀。
  • 添加定向棱柱调整指定词缀类别随机到其他类别词缀。
  • 橙装不能移除词缀，可以用这个办法把不要的词缀类别换成其他类别。

3）定向重洗

  • 添加定向棱柱，洗对应类别词缀。

4）移除词缀

  • 移除装备上不要的词缀，比如资源类换成伤害类。
  • 添加定向棱柱调整想移除的词缀类别。

5）嬗变物品

  • 类似之前赛季的圣化功能，给物品添加随机效果（有可能替换词条废掉装备），添加之后极大概率无法再次修改（无法洗词条/回火/锻造/附魔/打孔，但可以拆装宝石符文），所以必须放在最后一步。
  • 搞出大量做好前置工作（洗词条/回火/锻造/附魔/打孔）的底材，再去最终嬗变抽奖。900装等回火可以搞出星号词缀，好装备别随便回火就去嬗变了。
  • 如果是项链，可以先用库勒棱柱添加或更换额外的随机威能（不限部位通用池子抽取 ）。这个操作不会锁死装备，调整好了再去普通嬗变。补丁更新后，也可以调整项链自身的威能。
  • 熵能棱柱嬗变基本没用，不能出最好效果比如宝石特效且100%锁死，只能防止替换词缀，直接大量底材赌普通嬗变即可。除非是特别好的底材怕替换词缀，可以保守点。另外，神话暗金不会被替换词条，无需保底。
  • 嬗变效果不错的：全技能+4（看具体BD）、XX伤害增加（比如电系巫师加电伤）这个是独立增伤。
  • 因为普通嬗变极小概率不会锁死装备，有大量嬗变材料和底材就可以这样玩：大量900装等橙装（项链得先添加额外威能）直接拿去普通嬗变，只要次数够多必定能出想要的效果+不锁死，然后精炼词缀，再去魔盒调整其他词缀/附魔改威能/打孔。最后可以保守点加熵能棱柱再嬗变（防止替换词缀）或者干脆不再嬗变方便调整威能。

6）重置暗金威能

  • 随机修改暗金特效的数值，原始数值太低的有救了。
  • 暗金神符与非先祖暗金S14赛季开始也可以重置暗金威能。

2、物品塑形

1）三合一塑形

  • 3个同类型神符/封印 合成 1个新的同类型（词条随机）。
  • 3个同类型装备 合成 1个新的同类型（词条随机）。
  • 3个同样的符文 合成 1个新的同类型随机符文。
  • 嬗变失败的底材可以三合一变成新的底材。

2）回收暗金

  • 3个同样的暗金或暗金神符合成1个新的（词条随机）。
  • 3个装备都要最高等级，不然结果是3个里面最低等级。
  • 嬗变失败的普通暗金或神话暗金可以重生新的继续嬗变，比如3个军帽合成1个全新军帽，不会变成其他暗金。

3）升级为暗金

  • 白色装备升级为对应部位的暗金。
  • 记得底材装备等级要最高等级的，升级不会改变装备等级。

4）升级至传奇

  • 黄色装备升级成随机威能的橙色装备。
  • 前期威能缺乏的时候，可以有效获取所需威能，此时不看装备等级。
  • 可以添加类别棱柱调整威能类别。

5）重置套装符文

  • 把绿色套装符文随机更换成同套装的符文。
  • 套装不全的时候，可以用来补齐缺的符文。
  • 后期用来洗符文词缀，比如巫师需要每个符文带电冲等级。

6）制作暗金神符

  • 把先祖暗金装备做成对应的暗金神符（部分暗金除外）。
  • 每次需要消耗3个任意暗金神符，没用的暗金神符要留一些。
  • 暗金的词缀不会影响做出的暗金神符词缀。
  • 做出来的属于已制作（全身已制作只能装备一件）。

7）融合

  • 5个普通梦魇地下城符印 合成 1个升级梦魇地下城符印（连打三关那种）。
  • 5个相同蓝色贡品 合成 1个黄色同类贡品。
  • 5个相同黄色贡品 合成 1个橙色同类贡品。
  • 5个橙色军械贡品 合成 1个紫色神话军械贡品。
  • 5个相同首领战利品 合成 1个该首领池子里的随机暗金。
  • 5个相同巨型宝石 合成 1个赫拉迪姆同类宝石
  • 5个相同赫拉迪姆宝石 合成 1个无暇赫拉迪姆同类宝石

3、符文制作

用多余的符文合成各种橙色符文。

六、战利品过滤器

1、开启功能

  • 游戏菜单【选项】-【游戏设定】-【游戏选单启用战利品过滤器】。
  • 游戏菜单【选项】-【操作】-【给战利品过滤器设置一个快捷键】 。
  • 在游戏中通过快捷键开启/关闭战利品过滤器。

2、制作过滤器

前期不用过滤，后期简单示范：

1）添加一条神话暗金显示规则：

  • 规则1 稀有度为神话暗金。

2）添加一条暗金显示规则：

  • 规则1 稀有度为暗金。
  • 规则2 特性为先祖。
  • 规则3 指定要刷的暗金。
  • 规则4 指定必须带的词条，暗金无法洗全部词条，通过指定词条方便刷。

3）添加一条星号装备显示规则：

  • 规则1 稀有度为白色 蓝色 黄色 橙色。
  • 规则2 特性为先祖。
  • 规则3 带1~3条强化词缀（星号词缀），星号装备是做装底材，装备越好对底材要求越高。
  • 规则4 指定必须带的词条，不确定就不加。
  • 规则5 指定类型，不确定就不加。

4）添加一条升级威能显示规则：

  • 规则1 能量宝典升级匹配，显示能升级威能的橙装。

5）添加一条神符和封印显示规则：

  • 规则1 物品类型匹配，选择神符和封印。

6）添加一条隐藏规则放在最下面：

  • 内容为空（隐藏所有掉落）。
  • 规则从上到下匹配，走完上面所有规则匹配，最后隐藏剩下的。

在这个基础上根据自身需要调整。
